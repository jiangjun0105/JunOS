# Notes on Build Pipelines

Mostly a reminder to myself about where build time actually goes.

## Where the time goes

- Cold installs and uncached dependencies.
- Type-checking the whole graph instead of only what changed.
- Re-doing work that an incremental cache could have kept.

```bash
# the cheap wins, in order
pnpm install --frozen-lockfile
turbo run build --cache-dir=.turbo
```
