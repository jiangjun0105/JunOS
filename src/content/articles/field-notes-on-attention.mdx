# Field Notes on Attention

A few half-formed thoughts I keep coming back to. Nothing here is settled — these
are _field notes_, not conclusions.

## The one-sentence version

Attention is a **soft lookup**: every token writes a query, every token offers a
key, and the match scores decide whose values get mixed into whom.

> If you can phrase your problem as "who should look at whom, and how much?",
> there is probably an attention layer hiding inside it.
